Ring Counter
Simulation of a Ring Counter built using D flip-flops.

Start- Starts the simulation after the inputs are properly selected. Default speed is 1x. Select the speed through dropdown menu

Reset - Resets the simulation

Click on the green input bits to set their values.

This is a Ring Counter.

There is a clock and ori which can be started by the user but it's cycle cannot be controlled.

There are 3 outputs QA,QB,QC
